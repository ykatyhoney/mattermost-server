Keyboard shortcuts help you make a more efficient use of your keyboard when navigating Mattermost teams in a web browser or the desktop app.

<Tip>

See a list of available keyboard shortcuts any time by pressing <kbd>Ctrl</kbd> <kbd>/</kbd> on Windows or Linux, pressing <kbd>⌘</kbd> <kbd>/</kbd> on Mac, or using the `/shortcuts` slash command.

</Tip>
