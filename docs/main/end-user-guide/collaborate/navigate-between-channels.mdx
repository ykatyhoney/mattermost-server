## Navigate between channels

Using Mattermost in a web browser or the desktop app, you can navigate between channels by selecting the **Find channel** option in the channel sidebar, or by pressing <kbd>Ctrl</kbd> <kbd>K</kbd> on Windows or Linux, or <kbd>⌘</kbd> <kbd>K</kbd> on Mac. The Find channels screen also displays [member availability](/end-user-guide/preferences/set-your-status-availability) at a glance.

> ![Navigate between channels and review member availability.](/images/switch-channels.png)

## Return to recently viewed channels

Using a web browser or the desktop app, use the **History** arrows at the top of the sidebar to move back and forth through your channel history.

- Select the left arrow to go back one page.
- Select the right arrow to go forward one page.
