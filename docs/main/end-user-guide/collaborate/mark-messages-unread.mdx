---
title: "Mark messages as unread"
---
import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<PlanAvailability slug="all-commercial" />

If you read a message, but don't have time to address it right away, you can mark that message as unread. Marking a message as unread displays the channel as bold in the channel sidebar, and groups the message with all other unread messages.

<Tabs>
<TabItem value="web-desktop" label="Web/Desktop">

Hover over the message, select the **More** <img src="/img/ui/dots-horizontal_F01D8.svg" alt="Use the More icon to access additional message options." className="theme-icon" /> option, then select **Mark as Unread**.

![When you hover over messages, you can access more message options from the More icon.](/images/message-more.png)

![You can mark messages as unread to return to them later.](/images/mark-message-as-unread.png)

</TabItem>
<TabItem value="mobile" label="Mobile">

Long press a message, and then tap **Mark as Unread**.

![Tap and hold on a message to access the available options.](/images/mobile-mark-a-message-as-unread.gif)

</TabItem>
</Tabs>
